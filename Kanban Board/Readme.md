# Kanban Board Project

Yeh aik simple aur functional Task Management tool hai jo agile workflow (Kanban) ko follow karta hai.

## Key Features
- **Drag & Drop:** Tasks ko aik column se doosre mein smoothly move karein.
- **Auto-Count:** Har column (To-do, In Progress, Done) ke tasks ki tadaad real-time mein update hoti hai.
- **Data Persistence:** Local Storage ka use kiya gaya hai taake page refresh karne par aapka data save rahe.
- **Delete Functionality:** Tasks ko kisi bhi waqt remove kiya ja sakta hai.

## Technologies Used
- HTML5
- CSS3 (Custom variables aur transitions)
- JavaScript (Vanilla DOM Manipulation)

## How to Run
1. Repository ko clone karein.
2. `kanban-board` folder mein jayen.
3. `index.html` file ko kisi bhi browser mein open karein.
